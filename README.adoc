# Demo Launcher

Graphical User Interface for Launching i.MX Demos.

## Building

Before installing `Demo Launcher`, ensure all dependencies are installed.
All of them are common dependencies found in any `GNU/Linux distribution`;
package names will be different, but it shouldn't be difficult to search
using whatever package management tool that's used by your distribution.

### Required Dependencies

* https://www.gtk.org/[GTK+3] (3.22.30 or newer).

. Install the dependencies:
+
```console
~# apt install build-essential libgtk-3-dev
```
+
. Invoke `pkg-config` to check if the installation went well:
+
```console
~$ pkg-config --cflags gtk+-3.0
```

[CAUTION]
====
If the above step returns an error, goto <<bugs>>
====

### Build Commands

. Clone the repository:
+
```console
~$ git clone https://github.com/diegohdorta/demolauncher.git
~$ cd demolauncher
```
+
. Create the build directory:
+
```console
~/demolauncher$ mkdir build
~/demolauncher$ cd build
```
+
. Compile the source code:
+
[source,console]
----
~/demolauncher/build$ cmake ..
~/demolauncher/build$ make
----

## Bugs

. Find the `package config` (_.pc_) file and export its path as follows:
+
[source,console]
----
~$ find /usr -name "*gtk+-3.0*"
~$ export PKG_CONFIG_PATH="<GTK+3_PACKAGE_CONFIG_FILE_PATH>:/usr/share/pkgconfig"
----
+
. it would be probably at:
+
[source,console]
----
~$ export PKG_CONFIG_PATH="/usr/lib/x86_64-linux-gnu/pkgconfig:/usr/share/pkgconfig"
----
